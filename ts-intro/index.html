Práctica con Typescript

Instalación y comandos

Para iniciar proyecto con TS se usa en consola CMD el comando:
tsc -init que creará el archivo tsconfig.json

Para transpilar a javascript archivos ts se usa en consola CMD el comando tsc seguido del nombre del archivo y con -w de watch revisar /observar los cambios:
tsc index.ts -w

Este comando anterior creará un archivo .js y con esto haremos que se transpile siempre que hagamos un cambio en el código del archivo ts.

No olvidemos indicar en head o body del archivo html la ruta a este archivo js creado automáticamente para ver en la consola del navegador los cambios de forma simultánea

Usaremos Google Chrome y sus herramientas de desarrollo

El comando npm install instalará los node modules que tendremos que ignorar con el archivo .gitignore

CLASES en Typescript

Recordemos que tienen la ventaja de que en las clases podemos definir las propiedades de tipo:

  • privadas PRIVATE = sólo es visible dentro de la clase en la que está. Necesitaremos constructor
  • públicas PUBLIC = es visible fuera de la clase en la que está. Por defecto si no ponemos nada, las propiedades de la clase en la que está son públicas
  • estaticas STATIC

Según el alcance o visibilidad de los mismos cuando accedamos a ellos escribiendo Heroe.nombreReal

Veremos entre las opciones esta clase static al ser visible

Haremos la forma corta del código dentro del constructor sin el cuerpo entre llaves

  • //alterEgo: string;
  • //edad: number;
  • //nombreReal:number;
